HTML Elements

    HTML Elements
    • The container and its content together are called an element.
    • An HTML element is defined by a starting tag
    • If the element contains other content, it ends with a closing tag
    < <'p'>....<'/p'> is an HTML element, <'h1'>...<'/h1'> is another HTML element.
  • Some HTML elements don't need to be closed, e.g: <'img.../'>, <'hr /'> and <'br /'> elements, void elements.

Attributes

  • HTML elements can have attributes
  • Attributes provide additional information about an element
  • Attributes are always specified in the start tag
  • Attributes come in name/value pairs like:name="value"

Core Attributes

  • The Id Attribute
    • used to uniquely identify any element within an HTML page, two primary reasons for use:
      • to identify just that element and its content
      • distinguish between elements that have the same name.
