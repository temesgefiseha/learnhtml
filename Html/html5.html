Core Attributes...

  • The title Attribute
    • gives a suggested title for the element to identify just that element and its content
    • often displayed as a tooltip
  • The class Attribute
    • used to associate an element with a style sheet, and specifies the class of element
    • More on this when you learn CSS
  • The style Attribute
      allows you to specify Cascading Style Sheet (CSS) rules within the element
