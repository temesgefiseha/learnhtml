Lists

  • HTML provides simple and effective ways to specify lists in documents.
  • List types:
    • Unordered lists
    • Ordered lists
    • Definition lists

Unordered Lists

  • Bullet points, order doesn’t matter.
  • The <'ul'> tag, which is a block tag, creates an unordered list.
  • Each item in a list is specified with an <'li'> tag.
  • Atribute: type=“bullet type” {disc, circle, square}

Ordered Lists

  • Order of items is important.
  • Each item in the display is preceded by a sequence value, default is Arabic numerals. Atribute:type=“number type”{1, i, I, a, A}
  • The list is the content of the <'ol'> tag

Nested Lists

  • Any type list can be nested inside any type list
  • The nested list must be in a list item

    • Definition Lists (for glossaries)

    • Allow to specify a term and its definition or description
    • List is the content of the <'dl'> tag
    • Terms being defined are the content of the <'dt'> tag
    • The definitions themselves are the content of the <'dd'> tag
